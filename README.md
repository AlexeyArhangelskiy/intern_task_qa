# Тестовое задание на стажировку Avito

## Структура проекта

- `Task_1.md`: Результат выполнения задания №1
- `img/`: Скриншоты багов для задания №1
- `dataForTest/`: Данные в формате JSON для подмены ответа по API для тестирования разных значений
- `output/`: Результаты выполненных тестов, скриншоты
- `test.ts`: Исходный код тестов TypeScript
- `TESTCASES.md`: Тестовые сценарии
- `package.json`: Файл с зависимостями и скриптами
- `tsconfig.json`: Конфигурация TypeScript
- `jest.config.js`: Конфигурация Jest
- `BUGS.md`: Баг-репорт для задания №2

## Инструкция по выполнению тестов

### 1. Описание

Этот проект содержит тесты, написанные на TypeScript с использованием фреймворка для тестирования Jest и библиотеки для автоматизации браузерных тестов Playwright.

### 2. Установка зависимостей и запуск тестов

1. Клонирование проекта

- Нажмите на кнопку "Code" и скопируйте ссылку на репозиторий.
- Откройте терминал и перейдите в каталог, где вы хотите разместить проект.
- Выполните команду `git clone`, а затем вставьте скопированную ссылку.
- Перейдите в папку Task_2.

2. Настройка Node.js
   Перед запуском тестов убедитесь, что у вас установлен Node.js и npm. Если они не установлены, вы можете загрузить их с [официального сайта Node.js](https://nodejs.org/).

   После этого выполните следующую команду в терминале (убедитесь, что вы находитесь в директории проекта папка Task_2), чтобы установить все необходимые зависимости проекта:

```bash
npm install
```

3. Настройка тестов<br>
   Убедитесь, что ваш проект настроен для работы с Jest и Playwright. Проверьте конфигурационные файлы, такие как jest.config.js и package.json, чтобы убедиться, что все пути и настройки указаны корректно.

4. Запуск тестов<br>
   Для запуска тестов выполните следующую команду:

```bash
npm test
```
